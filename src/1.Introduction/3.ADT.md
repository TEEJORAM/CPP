# Abstract Data Type

## What is Data Type?
- Representation of data (how data is stored)
- Operations on data such as +, -, *, /, %, ++, and --

## Abstract
- Means hiding internal details
- For example: While performing operations such as +, - or *, do we need to know how they are performed in binary form inside main memory? **No**, because we are concerned only with declaring a variable and using it to perform these operations(Internal details are abstracted)

- For example: When we declare  `int x;`-> we know it's a primitive data type

- When both are combined and internal details are hidden, it can be used as **new data type** called as **Abstract Data Type**
